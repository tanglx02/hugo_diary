一、安装hugo#

windows用户可以先去官网下载hugo框架， hugo官网地址 ，

苹果用户有安装HomeBrew工具的话可以直接输入brew install hugo进行安装。

下载完成后检查是否安装成功，输入: hugo version，若出现版本信息则表示安装成功。

二、新建站点#

找到存放hugo站点的路径输入hugo new site myblog_public，myblog_public 是存放博客的文件夹名，完成后可以看到hugo目录

三、选择主题并下载#

官网下载主题 个人比较喜欢以下两款主题

  1. maupassant主题 飞雪无情的博客预览
  2. PaperMod 主题 Sulv’s Blog预览

四、部署主题和预览#

切换到hugo主目录，将主题下载到themes

git clone https://github.com/rujews/maupassant-hugo themes/maupassant

然后在Hugo的配置文件里config.toml(yaml,json)中进行如下配置。

baseURL = "/"
languageCode = "zh-cn"
title = "超级星小汤-tanglx"
theme = "maupassant"

添加一篇文章

hugo new posts/my-first-post.md

在主目录下编译博客

hugo

查看运行效果

hugo server -D

之后就可以通过浏览器输入http://localhost:1313访问我们的站点了

hugo博客的常用命令#

查看版本hugo version

新建站点 hugo new site hugoblog

访问本地站点

默认访问 hugo server -D 默认访问通过127.0.0.1:1313 自定义端口访问 hugo server -p 8080 自定义ip访问 hugo server --bind "0.0.0.0" -b "http://ip:1313"
